# 基于Python的OpenCV基础训练

通过几个小项目，新手能使用Python快速学习OpenCV，囊括图像处理的常用基本操作。另外还有一点涉及深度学习的项目

+ 几何变换：透视变换

+ 灰度变换：阈值处理、直方图处理、距离变换

+ 图像平滑：图像采样、图像滤波、傅里叶变换

+ 边缘检测：梯度处理、Canny边缘检测、几何检测、形状检测、角点检测

+ 形态学技术：腐蚀膨胀、开闭运算、Top-Hat

+ 特征分析：尺度空间、特征检测、特征匹配

+ 视频处理：背景建模、光流估计、目标跟踪

+ 深度学习：人脸检测与识别、EAST文本检测和识别、手写数字识别

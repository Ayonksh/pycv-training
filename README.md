# 基于python的opencv基础训练

通过几个小项目，新手能使用python快速学习opencv，囊括图像处理的常用基本操作

+ 灰度变换：阈值处理

+ 图像平滑：图像滤波

+ 边缘检测：梯度处理、Canny边缘检测、形状检测

+ 图像矫正：透视变换

+ 形态学技术：腐蚀膨胀、开闭运算、Top-Hat
